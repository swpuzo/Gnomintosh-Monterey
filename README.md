# Gnomintosh-Monterey
Gnomintosh Monterey is an customization suite for Linux. It transforms the GNOME desktop environment into a precise visual clone of Apple's macOS 12 Monterey.

# Features
* **Dock:** Floating bottom panel with corner rounding, background blur, and icon magnification.
* **Top Bar:** System status indicators on the right, active application global menu on the left.
* **Control Center:** iOS-style dropdown menu for quick Wi-Fi, Bluetooth, and brightness toggles.
* **Window Controls:** Traditional Apple "traffic light" buttons (red, yellow, green) on the top-left.
* **Aesthetics:** macOS Monterey vector icon packs, San Francisco fonts, and dynamic time-of-day wallpapers.

## How to install?
Install the extension from the "Extension" folder, download it. Then download the config from the "dconf" folder. and place it in your home directory. Enter the command: dconf load /org/gnome/shell/extensions/ < Gnomintosh_extensions.conf then download the wallpaper Then open the "logo menu" extension settings and download the logo of your choice from the "logo" folder and place it in the logos then download the icons and theme below in "Credits".
